# Invoke-CWCWake

## SYNOPSIS
Will issue a wake command to a given session.

## SYNTAX

```
Invoke-CWCWake -GUID <Guid[]> -Type <String> [<CommonParameters>]
```

## DESCRIPTION
Will issue a wake command to a given access or support session.

## EXAMPLES

### EXAMPLE 1
```
End-CWWake -GUID $GUID
```

Will issue a wake command to a given session.

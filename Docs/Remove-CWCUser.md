# Remove-CWCUser

## SYNOPSIS
Removes an internal user.

## SYNTAX

```
Remove-CWCUser [-User] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes the provided internal user.

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-CWCUser -User 'ctaylor'
```

Removes the 'ctaylor' user from Control.
